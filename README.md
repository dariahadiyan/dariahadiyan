## 📌 Featured projects

### [nyc-restaurant-investment-intelligence](https://github.com/dariahadiyan/nyc-restaurant-investment-intelligence)
End-to-end Snowflake pipeline analyzing 399K+ NYC restaurant inspection records. Raw → staging → mart layers with composite location opportunity scoring to identify optimal restaurant investment locations. Inspired by my family's roots in the NYC restaurant industry.

### [retail-sales-pipeline-databricks](https://github.com/dariahadiyan/retail-sales-pipeline-databricks)
Retail sales pipeline built on Databricks using medallion architecture (Bronze/Silver/Gold) with PySpark and Delta Lake for scalable ingestion, cleaning, and analytics-ready reporting.

### [cannabis-market-intelligence-public](https://github.com/dariahadiyan/cannabis-market-intelligence-public)
Public version of a production pipeline I built and run at work: automated weekly market intelligence in Python that pulls retail and competitor data, generates AI-powered insights via the Anthropic API, and delivers formatted HTML email reports on a GitHub Actions schedule — every Monday morning, no human required.

### [resume-tailor](https://github.com/dariahadiyan/resume-tailor)
LLM-based resume tailoring pipeline using the Anthropic API, with structural safeguards that prevent the model from inventing experience — it can only rephrase and reprioritize what's actually there.
